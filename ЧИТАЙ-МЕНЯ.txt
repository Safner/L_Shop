Папка участника A — только его файлы для сдачи / первого коммита.

Состав:
  package.json, tsconfig.json, eslint.config.mjs, .prettierrc, .gitignore
  src/index.ts          — Express, раздача public, заглушки /api/*
  public/index.html     — вёрстка страницы
  public/style.css      — стили
  public/app.js         — запросы к API (GET работают, POST/DELETE вернут 501 до B и C)

Запуск (из этой папки):
  npm install
  npm run dev
  Открыть http://localhost:3000 — каталог пустой, корзина пустая; кнопки «В каталог» / «В корзину» покажут текст из заглушек.

Дальше участники B и C заменяют заглушки в index.ts на роуты из shop-mini (или вливают общий проект).
