================================================================================
  Финальная версия проекта «Магазин» (L_Shop)
================================================================================

Эта папка — готовый к запуску проект: сервер Express, API товаров и корзины,
регистрация и вход (JWT), страницы /register и /shop.

Запуск (PowerShell):

  cd c:\finalgit\shopmini\final
  npm install
  npm run build
  npm start

В браузере: http://localhost:3000  (откроется регистрация)
Магазин: http://localhost:3000/shop

Разработка с перезапуском: npm run dev

Папки node_modules и dist здесь нет — появятся после npm install и npm run build.

Опционально: переменная окружения JWT_SECRET для продакшена.

================================================================================
